# 00-Home

This folder is the **entry point** for the Production Support Wiki. The `00-` prefix keeps it at the top of the file explorer so newcomers find orientation material first.

## Contents

| Note | Purpose |
|------|---------|
| [[Production Support Wiki]] | Main landing page — start here. Navigation, stats, and vault map. |
| [[Plugin Setup Guide]] | How to configure Obsidian (Dataview, Templates) for this vault. |
| [[Conventions and Usage Guide]] | Metadata schema, naming conventions, tagging taxonomy, Dataview rules. |

## What belongs here

- The homepage and top-level navigation
- Setup, onboarding, and convention documents
- Vault-wide reference material

## What does NOT belong here

- Incidents → `Incidents/`
- Runbooks → `Runbooks/`
- KB articles → `KB/`
- Policies → `Policies/`
- Templates → `Templates/`

Keep this folder small and orientation-focused. It is the front door, not a storage closet.
